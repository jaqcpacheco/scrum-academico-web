## 🚀 Sobre

Aplicação web desenvolvida para analisar dados do Trello e gerar métricas de desempenho de equipes ágeis, com base nas metodologias Scrum e Kanban.

O sistema realiza o processamento das tarefas para extrair indicadores como produtividade, eficiência, fluxo de trabalho (WIP) e backlog, permitindo uma visão estruturada do desempenho da equipe. Além disso, utiliza Inteligência Artificial para gerar insights automáticos, auxiliando na identificação de gargalos e na tomada de decisão.
