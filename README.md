📊 Scrum Acadêmico Web

Sistema web desenvolvido para análise de desempenho de equipes ágeis utilizando dados do Trello.

🚀 Sobre o projeto

O Scrum Acadêmico Web é uma aplicação que coleta informações de boards do Trello e transforma esses dados em métricas visuais e insights inteligentes, auxiliando na avaliação da produtividade e organização de equipes.

O sistema apresenta indicadores como:

Total de tarefas
Tarefas em andamento
Tarefas concluídas
Backlog
Produtividade
Distribuição de tarefas por membro

Além disso, o sistema conta com geração de insights com Inteligência Artificial, oferecendo análises mais profundas sobre o desempenho da equipe.

🧠 Diferencial

O principal diferencial do projeto é a integração com IA, que permite:

Análise automatizada dos dados
Identificação de padrões
Sugestões baseadas no fluxo de trabalho
🛠️ Tecnologias utilizadas
Frontend: React + Vite + Tailwind CSS
Backend: Node.js + Express
API: Trello API
IA: OpenAI API
⚙️ Como executar o projeto
1. Clonar o repositório
git clone https://github.com/seu-usuario/seu-repo.git
2. Instalar dependências

Frontend:

cd frontend
npm install
npm run dev

Backend:

cd backend
npm install
npm run dev
🔐 Configuração

Crie um arquivo .env na pasta do backend com as seguintes variáveis:

TRELLO_KEY=your_key
TRELLO_TOKEN=your_token
OPENAI_API_KEY=your_api_key
📌 Observações
O projeto está em desenvolvimento acadêmico (PDS)
A integração com IA pode ser ativada ou desativada via configuração
O sistema depende de boards do Trello para funcionamento
💼 Objetivo

Este projeto tem como objetivo demonstrar a aplicação de metodologias ágeis (Scrum) aliadas à análise de dados e Inteligência Artificial, criando uma solução prática e aplicável ao mercado.
